🐛 Troubleshooting

If widgets still have dark backgrounds:

Check browser console for theme loading errors. The gradients should now use CSS variables like var(--surface-1) instead of dark hex colors.

If popout doesn't work:

Look for the red debug text "Modal should be rendering: [widget-id]" at top-right. Check console for popout debug messages starting with "🔧 WidgetGrid: Popping out widget".

If preset buttons don't work:

Check console for messages starting with "🎯 LeftRail: Preset clicked" and "🎯 LeftRail: Preset switch result". Make sure you're clicking the actual preset buttons in the Layouts section.

If edit mode doesn't show controls:

Check console for "🔧 WidgetGrid: Toggling edit mode" messages. The "Edit Mode Active" panel should appear near the top of the page when edit mode is enabled.
