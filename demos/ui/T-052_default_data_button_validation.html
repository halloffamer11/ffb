🧪 Validation Instructions

⚠️ Prerequisites: Ensure the dev server is running and the React app is accessible.

Setup:

  1. Start dev server: npm run dev
  2. Navigate to http://localhost:5173
  3. Access Data Management from the navigation or go directly to http://localhost:5173/data-management

Test Steps:

  1. Access Data Management Page
    Expected: Data Management page loads with three-step process visible
  2. Select Any Data Source (Step 1)
    Expected: Click any data source card (e.g., "Custom CSV"). Step 2 should become visible.
  3. Locate Default Data Button
    Expected: In Step 2, see prominent "🚀 Load Default Data (Top 300 2024)" button with gradient styling above the file upload area, separated by "OR" divider
  4. Click Load Default Data Button
    Expected: Button shows "Loading..." state briefly, then success message appears and Step 3 becomes visible automatically
  5. Verify Data Preview (Step 3)
    Expected: Preview table shows 10 rows of player data with columns: id, name, team, position, byeWeek, injuryStatus, points, adp. Status message shows "Successfully loaded default data with 300 players!"
  6. Import Default Data
    Expected: Click "Import Data" button. Should show "Importing..." state, then success message. After 2 seconds, automatically navigate back to dashboard.
  7. Verify Data Import Success
    Expected: Back on dashboard, widgets should show the imported player data (300 players). Search widget should find players like "Lamar Jackson", "Ja'Marr Chase", etc.
  8. Test Error Handling (Optional)
    Expected: If demo file is missing, should show clear error message without breaking the page
