✅ T-048: VBD Column Fixes

Player Search Widget improvements: VBD calculation + column stability

🛠️ Fixes Implemented

❌ Before (Issues)

  • Data columns shifted right when hovering over players
  • VBD column was not sortable

✅ After (Fixed)

  • Column layout stable on hover/selection
