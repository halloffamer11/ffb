🔧 T-047: Data Import Connection Fix

Critical fix for connecting imported CSV data to React widgets

🐛 Root Cause Identified

Issue: Data was being saved to localStorage but React widgets weren't displaying it

Problem: Import bypassed the legacy store → React store bridge, breaking the data flow

Evidence: Console showed "✅ Data Import: Saved 300 players to localStorage" but widgets remained empty

🔧 Technical Fix Details

❌ Before (Broken Flow)

  1. CSV Import → localStorage directly
  2. React Store ← Legacy Store (empty)
  3. Widgets ← React Store (empty)
  4. Result: No data in widgets

✅ After (Fixed Flow)

  1. CSV Import → Draft Store.dispatch()
  2. Legacy Store ← IMPORT_PLAYERS action
  3. React Store ← Legacy Store sync
  4. Widgets ← React Store (with data)

Code Changes Made:

// OLD: Direct localStorage save (broken) storage.set('players', players); window.dispatchEvent(new CustomEvent('workspace:players-changed')); // NEW: Use draft store dispatch (working) draftStore.dispatch({ type: 'IMPORT_PLAYERS', payload: players });
